# Task 2: Student Digital Portfolio

### **Objective**
Establishing a professional "Digital Hub" to showcase academic and technical growth across key platforms.

### **Platforms Selected**
1.  **GitHub:** For version control and showcasing technical coding projects.
2.  **LinkedIn:** For professional networking, internships, and industry connections.
3.  **Stack Overflow:** For data science practice and community problem-solving.

### **Project Report (150-200 Words)**
For my foundational portfolio, I chose **GitHub, LinkedIn, and Stack Overflow**. GitHub is essential for my B.Tech journey as it acts as a live repository of my code and collaboration skills. LinkedIn serves as my professional storefront, where I have updated my education details, institution, and expected graduation year to attract internship coordinators. Finally, Kaggle allows me to engage with data science competitions.

**Reflection:**
Setting up these profiles made me realize that a digital presence is no longer optional; it is expected. Over the next four years, I plan to use GitHub daily to document my labs and personal projects, while using LinkedIn to share my "Learning in Public" journey. The most challenging aspect was writing a professional bio as a first-year student, but I focused on my current learning path and my role as a Digital Ambassador.
